  ### Main scoring function                                                     
  ```python                                                                     
  async def score_chunks(                                                       
      chunks: list[RetrievalResult],        # From existing retriever.py        
      uploaded_at_map: dict[str, datetime], # doc_id -> upload datetime         
      authority_map: dict[str, str],        # doc_id -> trust tag               
      min_confidence: float = 0.40,                                             
  ) -> list[ScoredChunk]:                                                       
  ``` {data-source-line="109"}                                                  
                                                                                
  Steps:                                                                        
  1. For each chunk, compute freshness_score using                              
  uploaded_at_map[chunk.doc_id]                                                 
  2. For each chunk, compute authority_score using authority_map[chunk.doc_id]  
  3. Compute agreement_score: for each chunk, count how many OTHER chunks in    
  the list have cosine similarity > 0.7 (use chunk embeddings already computed, 
  or recompute). Normalize to 0-1.                                              
  4. Compute composite_score using the formula above                            
  5. Filter out chunks where composite_score < min_confidence                   
  6. Sort remaining by composite_score descending                               
  7. Return list[ScoredChunk]                                                   
                                                                                
  ### Confidence summary helper                                                 
  ```python                                                                     
  def summarize_evidence_quality(scored_chunks: list[ScoredChunk]) -> str:      
      if not scored_chunks: return "none"                                       
      avg = sum(c.confidence.composite_score for c in scored_chunks) /          
  len(scored_chunks)                                                            
      if avg >= 0.80: return "high"                                             
      if avg >= 0.60: return "medium"                                           
      if avg >= 0.40: return "low"                                              
      return "insufficient"                                                     
  ``` {data-source-line="129"}
