  ``` {data-source-line="125"}                                                  
                                                                                
  Implementation:                                                               
  1. If mode == VECTOR: just do vector search, return top_k                     
  2. If mode == KEYWORD: just do BM25 search, return top_k                      
  3. If mode == HYBRID:                                                         
     a. Vector search: top_k * 2 candidates                                     
     b. BM25 search: top_k * 2 candidates                                       
     c. RRF merge (see below)                                                   
     d. Take top_k * 2 from merged                                              
  4. If rerank=True: apply cross-encoder reranking on candidates, take top_k    
  5. Return final RetrievalResult list                                          
                                                                                
  ### Reciprocal Rank Fusion (RRF):                                             
  ```python                                                                     
  def reciprocal_rank_fusion(                                                   
      vector_results: list[tuple[str, float]],  # (chunk_id, score)             
      bm25_results: list[tuple[str, float]],                                    
      k: int = 60  # RRF constant, typically 60                                 
  ) -> list[tuple[str, float]]:                                                 
      scores = {}                                                               
      for rank, (chunk_id, _) in enumerate(vector_results):                     
          scores[chunk_id] = scores.get(chunk_id, 0) + 1 / (k + rank + 1)       
      for rank, (chunk_id, _) in enumerate(bm25_results):                       
          scores[chunk_id] = scores.get(chunk_id, 0) + 1 / (k + rank + 1)       
      return sorted(scores.items(), key=lambda x: x[1], reverse=True)           
  ``` {data-source-line="151"}                                                  
                                                                                
  ### Cross-Encoder Reranking:                                                  
  - Use sentence_transformers.CrossEncoder with model                           
  "cross-encoder/ms-marco-MiniLM-L-6-v2"                                        
  - Load as singleton (same pattern as embedder)                                
  - rerank(question: str, candidates: list[RetrievalResult]) ->                 
  list[RetrievalResult]:                                                        
    - Pair (question, chunk_text) for each candidate                            
    - CrossEncoder.predict(pairs) → scores                                      
    - Sort candidates by cross-encoder score descending                         
    - Update result.score with cross-encoder score                              
  - NOTE: cross-encoder model is ~80MB, downloads on first use. It's            
  CPU-friendly.
