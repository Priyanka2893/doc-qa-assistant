  ## Detailed Requirements                                                      
                                                                                
  ### config.py                                                                 
  Use pydantic-settings BaseSettings. Read from .env file. Include:             
  - GROQ_API_KEY: str                                                           
  - GROQ_MODEL: str = "llama-3.3-70b-versatile"                                 
  - EMBEDDING_MODEL: str = "all-MiniLM-L6-v2"                                   
  - EMBEDDING_DIMENSION: int = 384                                              
  - QDRANT_HOST: str = "localhost"                                              
  - QDRANT_PORT: int = 6333                                                     
  - QDRANT_COLLECTION_NAME: str = "documents"                                   
  - CHUNK_SIZE: int = 500                                                       
  - CHUNK_OVERLAP: int = 100                                                    
  - TOP_K_RESULTS: int = 5                                                      
  - MAX_FILE_SIZE_MB: int = 50                                                  
  - APP_ENV: str = "development"                                                
  - LOG_LEVEL: str = "INFO"                                                     
  - ALLOWED_ORIGINS: str = "http://localhost:3000"                              
  Use model_config = SettingsConfigDict(env_file=".env")                        
  Expose a get_settings() function with @lru_cache()                            
                                                                                
  ### models.py                                                                 
  Create these Pydantic v2 models:                                              
  - ChunkSource(BaseModel): chunk_index: int, text_excerpt: str, score: float,  
  page_number: int | None                                                       
  - UploadResponse(BaseModel): doc_id: str, filename: str, chunk_count: int,    
  page_count: int, status: str = "success", ingestion_time_ms: int              
  - AskRequest(BaseModel): question: str (min_length=3, max_length=1000),       
  document_id: str, top_k: int = Field(default=5, ge=1, le=20)                  
  - AskResponse(BaseModel): answer: str, sources: list[ChunkSource], model:     
  str, tokens_used: int, doc_id: str                                            
  - DocumentInfo(BaseModel): doc_id: str, filename: str, chunk_count: int,      
  page_count: int, uploaded_at: str                                             
  - HealthResponse(BaseModel): status: str, qdrant: str, embedding_model: str,  
  version: str                                                                  
                                                                                
  ### database.py                                                               
  Use aiosqlite for async SQLite. Create a documents table:                     
  ```sql                                                                        
  CREATE TABLE IF NOT EXISTS documents (                                        
      doc_id TEXT PRIMARY KEY,                                                  
      filename TEXT NOT NULL,                                                   
      file_size_bytes INTEGER,                                                  
      page_count INTEGER,                                                       
      chunk_count INTEGER NOT NULL,                                             
      uploaded_at TEXT NOT NULL DEFAULT (datetime('now')),                      
      status TEXT NOT NULL DEFAULT 'ready'                                      
  );                                                                            
  ``` {data-source-line="111"}                                                  
  Expose: get_db() async context manager, insert_document(), list_documents(),  
  get_document(), delete_document()                                             
  SQLite file path: ../data/documents.db (relative to app/)                     
                                                                                
  ### services/parser.py                                                        
  Functions:                                                                    
  - extract_text_from_pdf(content: bytes) -> tuple[str, int]: Use pymupdf       
  (fitz). Return (full_text, page_count).                                       
  - extract_text_from_txt(content: bytes) -> tuple[str, int]: Decode UTF-8,     
  return (text, 1).                                                             
  - chunk_text(text: str, chunk_size: int, chunk_overlap: int) -> list[str]:    
  Use langchain_text_splitters.RecursiveCharacterTextSplitter.                  
  - parse_and_chunk(filename: str, content: bytes, chunk_size: int,             
  chunk_overlap: int) -> tuple[list[str], int]: Route to correct extractor by   
  file extension, then chunk.                                                   
  Raise HTTPException(400) for unsupported file types. Raise HTTPException(422) 
  for empty files.                                                              
  Supported extensions: .pdf, .txt                                              
                                                                                
  ### services/embedder.py                                                      
  Singleton pattern for SentenceTransformer model (load once, reuse):           
  - Use sentence_transformers.SentenceTransformer                               
  - Model: from settings                                                        
  - Method: encode_texts(texts: list[str]) -> list[list[float]] — use           
  batch_size=32, convert_to_tensor=False, show_progress_bar=False               
  - Method: encode_query(query: str) -> list[float]                             
  - Async wrapper functions for FastAPI compatibility using                     
  asyncio.get_event_loop().run_in_executor()                                    
  Log model loading time using structlog                                        
                                                                                
  ### services/vector_store.py                                                  
  Qdrant operations using qdrant-client async:                                  
  - init_collection(client, collection_name, dimension): Create collection if   
  not exists, use Distance.COSINE, VectorParams(size=dimension)                 
  - upsert_chunks(client, collection_name, doc_id, chunks, embeddings,          
  filename, page_numbers): Create PointStruct for each chunk with payload       
  {text, doc_id, filename, chunk_index, page_number}. Use UUID for point IDs.   
  - search_chunks(client, collection_name, query_vector, doc_id, top_k) ->      
  list[ScoredPoint]: Filter by doc_id using must filter. Return top_k results.  
  - delete_document_chunks(client, collection_name, doc_id): Delete all points  
  with doc_id filter.                                                           
  - count_collection(client, collection_name) -> int: Return total point count. 
  Use qdrant_client.AsyncQdrantClient for all operations.                       
                                                                                
  ### services/llm.py                                                           
  Groq API integration:                                                         
  - Use groq.AsyncGroq client                                                   
  - Function: generate_answer(question: str, chunks: list[str], model: str,     
  api_key: str) -> dict                                                         
    - Build messages with system prompt and context                             
    - System prompt: "You are an expert document analyst. Answer questions      
  based ONLY on the provided document excerpts. If the answer is not in the     
  context, say 'I couldn't find information about that in the document.' Never  
  make up information."                                                         
    - User message: include numbered excerpts then question                     
    - Return: {answer: str, tokens_used: int, model: str}                       
  - Handle groq.RateLimitError -> HTTPException(429)                            
  - Handle groq.APIError -> HTTPException(502)                                  
                                                                                
  ### routers/health.py                                                         
  GET /api/v1/health                                                            
  - Check Qdrant connectivity (call count_collection)                           
  - Check embedding model loaded                                                
  - Return HealthResponse                                                       
  - If any check fails, status = "degraded" but still return 200                
                                                                                
  ### routers/documents.py                                                      
  POST /api/v1/documents/upload                                                 
  - Accept: UploadFile                                                          
  - Validate: file size < MAX_FILE_SIZE_MB, extension in [.pdf, .txt]           
  - Generate doc_id = str(uuid.uuid4())                                         
  - Call parse_and_chunk()                                                      
  - Call encode_texts() on chunks                                               
  - Call upsert_chunks()                                                        
  - Insert into SQLite                                                          
  - Return UploadResponse with ingestion_time_ms measured via                   
  time.perf_counter()                                                           
                                                                                
  GET /api/v1/documents                                                         
  - Query SQLite for all documents                                              
  - Return list[DocumentInfo]                                                   
                                                                                
  DELETE /api/v1/documents/{doc_id}                                             
  - Delete from Qdrant (delete_document_chunks)                                 
  - Delete from SQLite                                                          
  - Return {"status": "deleted", "doc_id": doc_id}                              
                                                                                
  ### routers/qa.py                                                             
  POST /api/v1/qa/ask                                                           
  - Accept: AskRequest                                                          
  - Verify document exists in SQLite (404 if not)                               
  - Embed question                                                              
  - Search Qdrant with doc_id filter                                            
  - Extract chunk texts from results                                            
  - Call generate_answer()                                                      
  - Build sources list from scored results                                      
  - Return AskResponse                                                          
                                                                                
  ### main.py                                                                   
  - Create FastAPI app with title, description, version                         
  - Add CORS middleware with allowed origins from settings                      
  - Include all routers with prefix /api/v1                                     
  - Lifespan context manager:                                                   
    - On startup: init Qdrant collection, load embedding model, create SQLite   
  DB                                                                            
    - Log startup/shutdown with structlog                                       
  - Add request logging middleware                                              
  - Use structlog for all logging                                               
  - Create data/ directory if not exists
