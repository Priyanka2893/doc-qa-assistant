  ### Step 2: New service services/deduplicator.py                              
                                                                                
  #### Hash-based deduplication (exact)                                         
  ```python                                                                     
  def deduplicate_exact(chunks: list[str]) -> tuple[list[str], int]:            
      """Remove exact duplicate chunks by SHA256 hash.                          
      Returns (unique_chunks, removed_count)"""                                 
      seen = set()                                                              
      unique = []                                                               
      for chunk in chunks:                                                      
          h = hashlib.sha256(chunk.encode()).hexdigest()                        
          if h not in seen:                                                     
              seen.add(h)                                                       
              unique.append(chunk)                                              
      return unique, len(chunks) - len(unique)                                  
  ``` {data-source-line="202"}                                                  
                                                                                
  #### Semantic deduplication (near-duplicate)                                  
  ```python                                                                     
  async def deduplicate_semantic(                                               
      chunks: list[str],                                                        
      embedder,                                                                 
      similarity_threshold: float = 0.95                                        
  ) -> tuple[list[str], int]:                                                   
      """Remove near-duplicate chunks using cosine similarity.                  
      Uses a greedy approach: keep the first chunk, remove any later chunk      
      with cosine similarity > threshold to any kept chunk.                     
      Returns (unique_chunks, removed_count)"""                                 
  ``` {data-source-line="215"}                                                  
  Algorithm:                                                                    
  1. Encode all chunks in one batch call                                        
  2. Build a list of "kept" chunks starting with chunk[0]                       
  3. For each subsequent chunk: compute cosine similarity to ALL kept chunks    
  4. If max similarity to kept chunks >= threshold: skip (it's a                
  near-duplicate)                                                               
  5. Otherwise: add to kept list                                                
  6. Return kept chunks and removed count                                       
                                                                                
  Only run semantic dedup if len(chunks) > 1. Skip if len(chunks) <= 3 (not     
  worth the overhead).                                                          
  Log the removed counts at INFO level.
